# Dual-Material Filament for Miniaturizing Soft Pneumatic Actuators via FDM

## Project Overview

**Degree:** BEng Mechanical Engineering – Honours Dissertation  
**Author:** Henry-Mc-Engineering  
**Date:** 2026  

This repository documents a novel dual-material filament process for miniaturizing soft pneumatic actuators using Fused Deposition Modelling (FDM) technology.

## Research Objectives

- Develop an innovative dual-material filament extrusion technique
- Demonstrate miniaturization of soft pneumatic actuators
- Validate the FDM process for soft robotics applications
- Compare performance metrics with traditional manufacturing methods
